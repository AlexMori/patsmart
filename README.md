# PATSmart: Lo Sportello Digitale della Provincia di Trento

**PATSmart** è una piattaforma web innovativa progettata per semplificare l’accesso al vasto ecosistema di servizi e strutture della **Provincia Autonoma di Trento**. Il progetto trasforma i dati aperti (Open Data) istituzionali in un'interfaccia utente fluida, moderna e orientata al cittadino.

---

### Caratteristiche Principali

* **Accesso Istantaneo**: Grazie a un sistema di **serverless caching** su Netlify, il sito elimina i tempi di attesa delle API istituzionali, offrendo un caricamento dei dati immediato e aggiornato automaticamente ogni 6 ore.
* **Navigazione Intuitiva**: Gli utenti possono esplorare l'organigramma gerarchico della PAT o consultare il catalogo dei servizi attraverso un potente sistema di filtraggio per **argomento**, **destinatario** (cittadini, imprese, enti) e **struttura** di riferimento.
* **Geolocalizzazione**: Una mappa interattiva integrata permette di individuare rapidamente gli uffici sul territorio, fornendo indicazioni stradali dirette tramite Google Maps.
* **Design Responsive**: Un'interfaccia pulita, sviluppata con Tailwind CSS, garantisce una consultazione ottimale da qualsiasi dispositivo, sia desktop che mobile.

---

**PATSmart** non è solo un aggregatore, ma un ponte digitale che rende l'amministrazione pubblica più vicina, trasparente e veloce.
